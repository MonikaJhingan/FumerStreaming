## Fumer Streaming- Video Library app

- This app is like a video streaming app Youtube.

## Features of the app 

- Home Page
- Video Listing Page
- Playlist Management
- Like/Dislike
- Watch Later
- History
- Authentication
- Sign-up
- Login


## Technology/Language Used

- CSS
- React JS
- React Router v6
- Mockbee
